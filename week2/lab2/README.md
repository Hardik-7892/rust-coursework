# Rust Structs & Networking Exercises

A collection of Rust exercises covering:

* Custom structs (`Book`)
* Vectors and iteration
* Network interface discovery
* IPv4 address classification (Classes A–E)

## Dependencies

```bash
cargo add get_if_addrs
```

## Run

```bash
cargo run
```

## Output

The program demonstrates:

1. Creating and accessing struct fields
2. Calculating the total price of a collection of books
3. Listing local network interfaces and IPv4 addresses
4. Determining the class (A–E) of each IPv4 address
